	1.	包结构说明:
	
	主包:com.nju.se.team.mota
	data              - 负责游戏数据存取及持有
	editor            - 编辑器, 包含UI元素及编辑器响应逻辑 
		dnd           - 内容拖拽(DragAndDrop)模块
		uielem        - UI列表元素
	game              - 游戏模块
		unit          - 游戏单位
		util          - 游戏内容所需的一些基本类(如动画类)
	temp              - 临时包, 包含一些临时(非计划性)测试
	util              - 基本类,工具类
	
************************************************************
************************************************************
	2.	项目基本说明
	
	内容: 编辑器 + 游戏本体
	
	1. 资源统一管理:res.list, 以ini格式存储
	2. 游戏需采用脚本系统(暂时采用伪代码表示), Linkeo想用JavaScript
	3. 游戏单位定义采用JSON存储, 单位符合Bean标准
	4. 编辑器地图编辑部分采用监听器式设计
